# Logs table

- Show top 20 logs in a table with react-table.
- Fetch this information on the client thorugh the new resource created by the BFF.
- Show an skeleton loader to reserve space, avoid layout content shifts.
- We are focusing on the very basic functionality, which means we do not want to add any additional complexity supported by react-table.
- Use Shadcn table component. Add it to the foundation components. Use ShadCN MCP to install it.

## Design choices

- Create the proper architectural artifacts to respect the layers of the frontend.
